Show WooCommerce variable products variations as table with filters and sorting instead of normal dropdowns.

== Description ==

Woo Variations Table replace normal WooCommerce variations dropdowns by professional table with filters by attributes and keywords.

You can control what columns to show in the variations table from settings `WooCommerce -> Woo Variations Table`.

== Installation ==

After downloading the ZIP file,

1. Log in to the administrator panel.
2. Go to Plugins Add > New > Upload.
3. Click "Choose file" ("Browse") and select the downloaded zip file.
4. You can control what columns you would show in variation table from WooCommerce -> Woo Variations Table.
